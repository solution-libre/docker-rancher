## Description

Docker-compose setup for starting [Rancher](https://rancher.com/).

## Setup

```sh
cd /opt
git clone https://github.com/solution-libre/docker-rancher.git rancher
cd rancher
```

Change the hostname value in `.env`.

## Usage

```sh
cd /opt/rancher
docker-compose up -d
```
